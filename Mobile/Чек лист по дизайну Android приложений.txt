 Navigation
- Приложение поддерживает стандартную навигацию с помощью кнопки «Назад»
- Приложение правильно сохраняет и восстанавливает состояние пользователя 
или приложения
- Приложение сохраняет состояние пользователя или приложения при выходе 
из переднего плана и предотвращает случайную потерю данных из-за обратной 
навигации и других изменений состояния
- Возвращаясь к приложению, оно должно возобновить сохраненное состояние
из которого пользователь прервал приложение
- Возобновление информации должно действовать:
- если прошло несколько минут – возобновить как можно 
больше информации
Notifications.
- Уведомления не используются для перекрестного продвижения 
или рекламы другого продукта, так как это строго запрещено Play Store
- Для приложений для обмена сообщениями, социальных приложений и разговоров:
- Используйте MessagingStyle уведомлений для разговоров
UI and Graphics
- Приложение должно поддерживать 
горизонтальное и вертикальное
положение, если это возможно
- Приложение быстро изменяет 
ориентации не изменяя свой статус
 Visual quality
- Графика,тексты, изображения и другие 
 UI елементы не должны быть размыты/искажены
- Текст не обрезается, соблюдаются пропорции в расстоянии 
текста и окружающих элементов
- Приложение должно поддерживать темную тему
- Контраст между текстом и фоном
- 3.0:1 для крупного текста, графики
- 4.5:1 для текста меньше 18 пунктов, или полужирный текст 
меньше 14 пунктов
Functionality.
Audio:
- Если юзер запускает аудио, приложение должно за 1 
секунду:
- начать играть аудио
- выводится визуальный индикатор того, что воспроизводится аудио
- Приложение должно обрабатывать запросы других приложений на аудиофокус
Media
- Если приложение воспроизводит видео, оно должно 
пожжерживать режим picture-in-picture
- Приложение должно использовать Android Sharesheet при обмене контентом
Privacy & security.
- Приложение должно использовать минимальное количество 
разрешений для работы
- Приложение должно запрашивать доступ к ensitive data (таких 
как SMS, Call Log, или Location)
- Персональные данные не должны логироваться
- Ввесь трафик должен высылаться через SSL
